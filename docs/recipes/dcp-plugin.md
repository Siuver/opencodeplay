# DCP Plugin Recipe

This recipe tracks the desired `opencode-dynamic-context-pruning` companion plugin.

## Current Status

- Package: `@tarquinen/opencode-dcp`
- Version: `3.1.11`
- License: `AGPL-3.0-or-later`
- Status: planned and disabled in the pinned-artifact backend.

## Do Not Enable Until

- The npm tarball checksum is replaced with a real SHA-256.
- The offline opencode/Bun plugin cache or local install flow is documented.
- Validation inside `opencode` is tested.

## Online Shape

Upstream documents an online plugin install shape, but unattended setup must use a pinned version and verified artifact. Do not use `latest` in this repo's automated paths.
